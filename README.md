# The accelerated tensor Kaczmarz algorithm with adaptive parameters for solving tensor systems

:computer: This repository contains the MATLAB code for tAKSHBM introduced in the following paper 

```html
Y. M. Liao, W. Li, D. Yang. The accelerated tensor Kaczmarz algorithm with adaptive parameters for solving tensor systems. Applied Numerical Mathematics, 202, 100-119, 2024. 
```

:link: DOI: https://doi.org/10.1016/j.apnum.2024.04.016 

### Preparation 

* **Windows 64bit** 
* **Example usage**:

```MATLAB
Please unzip the project and add it to the MATLAB path first, and then run the demo01-demo04.
```
